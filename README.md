# Image_classification
# Image Classification with Limited Labels

## Project Overview

This project implements image classification using a flower dataset.

## Dataset Classes

- Daisy
- Dandelion
- Rose
- Sunflower
- Tulip

## Technologies Used

- Python
- NumPy
- Matplotlib
- Pillow (PIL)
- Scikit-Learn

## Workflow

1. Load Images
2. Preprocess Images
3. Split Dataset
4. Train Model
5. Evaluate Accuracy
6. Predict Flower Categories

## Outcome

Successfully implemented an image classification model using machine learning.
